# image_process
图像处理，加噪去噪，滤波，均衡化,霍夫变换,hog特征,canny算子,lbp特征, 图像形态学操作

# tips
## imageWork1
两个文件夹里是已经处理好的图像数据
`filter.py`是对图像进行加噪和滤波操作的
`pic_average.py`是对图像进行均衡化处理的
`image_process.py`也是图像滤波
`morphology.py`是图像形态学操作

## imageWork2
`result`文件夹里是已经处理好的文件
`canny.py`是canny算子
`hough.py`是图像霍夫变换检测直线


## imageWork3

`result`文件夹里是已经处理好的文件
`HOG.py`是hog特征
`LBP.py`是lbp特征
